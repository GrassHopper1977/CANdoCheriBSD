Netronics CANdo Linux SDK

Check the website www.cananalyser.co.uk for the latest version.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This folder contains a collection of small programs written in
various programming langauges that all use the libCANdo.so shared
library to communicate with the CANdo devices.

The libCANdo.so shared library must be compiled & installed on the
PC or embedded computer module, before these programs can be
executed.
